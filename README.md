# 📚책 검색 서비스 Bibliophile's Choice

카카오 책 검색 api를 활용한 책 검색 서비스입니다.

Bibliophile는 ‘애서가’라는 의미로, 책을 사랑하는 사람들의 차분하고 모던한 이미지를 담아낸 디자인과 함께 원하는 책을 찾아볼 수 있습니다.

메인 페이지의 상단 검색바를 통해 책을 찾아볼 수 있고, 키워드와 작가에 따라 필터링 된 책을 볼 수 있습니다.
